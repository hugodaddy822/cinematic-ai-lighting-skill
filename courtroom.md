# Courtroom Example

## User Request

```text
使用 cinematic-ai-lighting skill，把这张法院里的法官画面做得更有电影感。
保留人物身份、构图、服装和画幅比例。
```

## Prompt

```text
在不改变原图比例、构图、人物身份、服装和镜头角度的前提下，
为法院场景添加真实电影感布光。
从画面右后方的高位窗户打入一束侧逆光，
在人物肩部、头发边缘和桌面形成自然轮廓光；
空气中有轻微尘埃和薄雾，让光束可见但不过度遮挡主体；
人物正面保留非常柔和的低强度补光，使五官和表情仍然清晰；
整体为低调高反差，背景更暗，主体与背景明显分离，
木质桌面有细腻高光，阴影有层次，画面呈现严肃、戏剧化、真实电影质感。

避免：改变人物长相、改变画幅比例、过曝、强舞台灯、过度雾化、卡通化、塑料皮肤、随机新增灯具。
```
