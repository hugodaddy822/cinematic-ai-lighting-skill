# Car Interior Example

## User Request

```text
使用 cinematic-ai-lighting skill。
场景是夜晚车内人物特写，我想要电影感侧逆光、窗外霓虹、柔和正面补光。
```

## Prompt

```text
保持原图人物身份、车辆内饰、构图和镜头角度不变。
车窗外有蓝紫色霓虹灯作为主要环境光，
从人物侧后方形成冷色轮廓光；
仪表盘和中控屏给人物面部提供低强度柔和补光，
脸部可读但不过亮；
车窗上有轻微雨痕和城市反光，
暗部保留细节，背景虚化，整体呈现夜晚都市电影感。

避免：改变车型、改变人物长相、过度赛博、霓虹太脏、脸部死黑、整体过曝。
```
